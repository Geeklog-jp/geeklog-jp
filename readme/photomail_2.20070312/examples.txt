
(例1)既定値は（「受信メールサーバーの設定」以外修整しなかった場合）
   ゲストユーザの投稿はできません。
   登録ユーザの投稿はできます。

(例2)ゲストユーザの投稿を認めたい時
    $this['anonymous'] = 1  にする
    ゲストユーザの投稿は承認待ちファイルに登録されます。
    登録ユーザの投稿は通常の記事ファイルに登録されます。

(例3)許可するグループの設定,許可するメールアドレスの設定で設定
    した内容以外の投稿を認めたくない時
    $this['acceptonly'] = 1　にする
    登録ユーザであっても設定されてない場合は投稿できません。

(例4)STORY ADMIN グループの投稿は、即反映(下書きではなく)
    その他の登録ユーザの投稿は下書にしたい
    ゲストユーザの投稿は認める場合
    $this['anonymous'] = 1  にする
    $this['acceptonly'] = 0　にする
