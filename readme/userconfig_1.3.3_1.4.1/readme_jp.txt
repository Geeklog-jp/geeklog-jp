このプラグインのご利用方法

インストール

１．
念のため、config.php、lib-common.php、データベースのバックアップをとっておいて
ください。

２．
展開されたファイルのうち、publicディレクトリのファイルは公開ディレクトリに、
privateディレクトリのファイルは非公開ディレクトリに、それぞれ配置してください。

３．
(非公開ディレクトリ)/data/ ディレクトリ　のパーミッションを書込&実行許可にしてください。
（例：707）

４．
Geeklog本体に付属するconfig.phpの最後の「?>」の前の行に、同梱ファイル「add_to_config.php」の内容をコピーしてください。

５．
Geeklogの管理者メニューの「プラグイン」からインストールしてください。




アンインストール

１．
Geeklogの管理者メニューの「プラグイン」からアンインストールしてください。

２．
インストール時に配置したファイルとディレクトリを削除してください。




バージョンアップ

１．
現在の設定をバックアップしてください。

２．
古いコンフィギュレーションエディタをアンインストールしてください。

３．
新しいコンフィギュレーションエディタを上書きしてください。

４．
新しいコンフィギュレーションエディタをインストールしてください。

５．
バックアップに戻してください。

６．
念のため、(非公開ディレクトリ)/data/userconfig_bak.php　をローカルに退避してください。

７．
現在の設定をバックアップしてください。




使い方

Geeklogの管理者メニューの「コンフィギュレーション」からご利用ください。
