Install

step.1
Please make back up files ,config.php and lib-common.php.

step.2
Upload local public directory to server public directory,
local private directory to server private directory.

step.3
Change permission executable private/data/ directory.
(exsample:707)

step.4
Install this plugin from Geeklog admin's menu plugin.

step.1
Uninstall this plugin from Geeklog admin's menu plugin.

step.2
Remove all files installed.
