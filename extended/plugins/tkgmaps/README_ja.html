Geeklog tkgmapsプラグインの使い方:

初期設定

tkgmapsプラグインは，自動タグプラグインの関数機能を利用して動作します。

そのため，あらかじめ自動タグプラグインのconfig.phpで，関数機能を利用できるよう，設定を変更しておく必要があります。

修正前：

$_AUTO_CONF['allow_php'] = 0;

修正後：

$_AUTO_CONF['allow_php'] = 1;

　

運用方法：

自動タグを記事，静的ページ，ブロックで次のように記述して利用します。

[maps:base ＜住所または　緯度,経度＞ width:＜数字＞ height:＜数字＞ zoom:＜数字＞ margin:＜数字＞]
[maps:point ＜住所＞ title:＜タイトル＞ icon:＜アイコン種類　pngファイル名(拡張子なし)orURL＞ iconsize:＜アイコンサイズ　x,y＞ iconshadow:＜影アイコン　URL＞ iconshadowsize:＜影アイコンサイズ　x,y＞ iconanchor:＜アンカー位置　top,left＞ infowindowanchor:＜インフォウィンドウ位置　top,left＞ info:＜インフォメーション＞ ＜バルーン内自由なHTMLの記述＞]
・・・・複数pointを記述

[maps:show]
[maps:display]

width, height, zoom, margin, title, icon, png, infoなどの属性は省略できます。

icon:blue-pushpin　icon:green　icon:picnic　のようにアイコンをひとつひとつ変更できます。

参考：
http://maoxiong.seesaa.net/article/45950146.html
png画像がいろいろ用意されています。.pngを除いたファイル名を指定することで，
このなかのアイコンをどれでも利用できます。

住所の代わりに，34.297306,132.31814　のように緯度・経度で記述することもできます。

自動タグの記述中に改行を入れないでください。アドバンストエディタを利用すると自動的に改行が入る場合がありますので,注意してください。
