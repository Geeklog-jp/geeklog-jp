FCKeditor v2.5 に対して
　MGメディアブラウザープラグイン（mg Media Browser - FCKeditor Plugin v1.5.0）
を追加したパッケージとなっています。

使い方：

メディアギャラリを利用しているサイトでは，ツールバーのMGアイコンを利用して
メディアギャラリのアイテムを配置できます。

ツールバーの内容はmyconfig.jsで自由に変更できます。
MGアイコンが不要であればmyconfig.jsから削除して利用してください。

インストール方法：
これらのファイルをfckeditorにアップロードして差し替えるだけです。

日本語化とMGメディアブラウザープラグインの追加箇所：

　fckconfig_org.js メディアギャラリプラグインを追加しないオリジナル
　fckconfig.js MGメディアブラウザープラグインの追加
　myconfig_org.js メディアギャラリプラグインを追加しないオリジナル
　myconfig.js MGメディアブラウザープラグインの追加
　editor/plugins/mediagallery/langs/english.js MGアイコン日本語翻訳
　editor/plugins/mediagallery/langs/english.php MGアイコン日本語翻訳
　editor/plugins/mediagallery/langs/english_org.js MGアイコンオリジナル言語ファイル
　editor/plugins/mediagallery/langs/english_org.php MGアイコンオリジナル言語ファイル
　editor/filemanager/connectors/php/config.php サーバーブラウズ不具合対策
