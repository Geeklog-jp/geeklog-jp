nmoxqrblock Plugin

Configuration

The configuration of the nmoxqrblock plugin can be changed from the Configuration admin panel:

Main Settings

Variable Default Value Description
image_type PNG The image type of QR code
ecc_level M Error correction level.
  • Level L - 7% of codewords can be restored
  • Level M - 15% of codewords can be restored
  • Level Q - 25% of codewords can be restored
  • Level H - 30% of codewords can be restored
module_size 2 Image module size
