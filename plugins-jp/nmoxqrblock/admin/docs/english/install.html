Install/Uninstall instruction for the Geeklog nmoxqrblock plugin

In the following descriptions

  • <geeklog_dir> is the directory where the system db-config.php file resides
  • <public_html> is the directory where the lib-common.php file resides
  • <admin> is the directory where the administration files reside (usually, under <public_html>)

INSTALL:

  1. Uncompress the nmoxqrblock plugin archive while in the <geeklog_dir>/plugins directory. The archive will create a directory called nmoxqrblock in the plugins directory.
  2. Create the public_html and admin directories. Under your html directory (usually <public_html>) create a directory called nmoxqrblock. Under your <admin>/plugins/ directory create a directory called nmoxqrblock.
  3. Change to your <geeklog_dir>/plugins/nmoxqrblock/ directory. Copy the files in the admin directory to the <admin>/plugins/nmoxqrblock/ directory your created in step 3. Copy the files and directories in the public_html directory to the <public_html>/nmoxqrblock/ directory your created in step 3.
  4. Log in to your Geeklog as a root user and install the plugin in the plugin editor.

UNINSTALL:

  1. Uninstall the plugin in the plugin editor.
  2. Delete the three plugin directories created in the install process: <geeklog-dir>/plugins/nmoxqrblock/, <public_html>/nmoxqrblock/ and <admin>/plugins/nmoxqrblock/

That's it.
